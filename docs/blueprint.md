# **App Name**: Ceasarion

## Core Features:

- Browser & Adblocker Detection: Accurately identify user browser type and detect the presence of adblocker software to inform ad serving strategy.
- Dynamic Ad Rendering: Seamlessly display the selected Adsterra advertisement format directly on the landing page, adapting to various device and browser characteristics.
- Performance Analytics Dashboard: Track and display essential metrics suchs as ad impressions and clicks per ad type and browser, providing insights for optimization.

## Style Guidelines:

- Primary brand color: A deep, refined indigo (#3B248F), symbolizing digital sophistication and efficiency. Used for headings and interactive elements.
- Background color: A very light, almost white lavender (#F4F1FA), providing a clean, airy canvas that highlights content and ads.
- Accent color: A vibrant sky blue (#56ABED), used sparingly for calls-to-action and key visual cues to provide energetic contrast.
- All text uses 'Inter', a grotesque-style sans-serif, for a modern, objective, and neutral look that is highly legible across all contexts.
- Utilize minimalist, modern line icons that suggest data flow, analytics, and connectivity, maintaining a clean and professional appearance.
- Implement a clean, content-first layout with generous white space. Use a modular, card-based approach for settings and statistics to ensure readability and easy navigation on various screen sizes.
- Incorporate subtle, quick-loading animations for elements like navigation transitions, button hovers, and ad display changes, enhancing user engagement without being distracting.
